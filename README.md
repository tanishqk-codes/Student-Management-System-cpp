## Advanced Student Management System (C++)

A console-based Student Management System built using C++.

## Features

- Login System
- Add Student
- Search Student
- Update Student
- Delete Student
- CGPA Calculation
- Student Ranking
- Statistics
- File Handling
- Marksheet Export

## How to Run

# Method 1: Online Compiler

1. Open Programiz C++ Compiler
2. Copy the code from "main.cpp"
3. Paste the code
4. Click Run

# Method 2: VS Code / Dev C++

1. Download the project
2. Open "main.cpp"
3. Compile and Run the code
